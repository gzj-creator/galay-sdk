## 本页回答什么

- `AsyncFile` 与 `AioFile` 的边界是什么

## 当前稳定事实

- `AsyncFile` 只在 `USE_KQUEUE` / `USE_IOURING` 下公开
- `AioFile` 只在 `USE_EPOLL` 下公开，且 `O_DIRECT` 读写要求对齐缓冲区
- 主干页已经承接了“公开 API / 用法 / 平台边界”的主要说明，本页只保留定位和锚点
