# Galay-RPC 文档总览

本目录的 Markdown 页面以仓库真实文件为准，推荐按下面顺序阅读：

1. [00-快速开始](00-快速开始.md) - 构建要求、依赖、首次运行命令
2. [01-架构设计](01-架构设计.md) - 协议、模块、运行时协作关系
3. [02-API参考](02-API参考.md) - 与公开头文件对齐的接口说明
4. [03-使用指南](03-使用指南.md) - 常见接入方式与编程模式
5. [04-示例代码](04-示例代码.md) - 仅收录能映射到真实 `examples/` / `test/` 文件的示例清单
6. [05-性能测试](05-性能测试.md) - 真实 benchmark target、命令和复现要求
7. [06-高级主题](06-高级主题.md) - 高级模式与应用层扩展说明
8. [07-常见问题](07-常见问题.md) - 故障排查与边界条件

## 文档真相来源

当文档描述与仓库内容冲突时，使用以下优先级：

1. 公开头文件与导出 target
2. 实现行为
3. `examples/`
4. `test/`
5. `benchmark/`
6. Markdown 文档

## 页面约定

- `04-示例代码` 只列出能直接命中真实文件与 target 的可验证示例。
- `05-性能测试` 只保留真实 benchmark target、参数和复现链路；未经重新复现的性能数字不会作为当前事实陈述。
- `06-高级主题` 同时包含两类内容：
  - **源码已提供能力**：可直接在公开头、示例或 benchmark 中找到依据。
  - **说明性模式**：用于展示集成思路，不代表仓库内置了对应实现或第三方依赖。
