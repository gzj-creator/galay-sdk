# 00-快速开始

本页只保留与当前仓库真实文件、真实 target、真实公开头一致的最短路径。

## 前置要求

- CMake `>= 3.22`
- `C++23` 编译器
- `spdlog`
- `galay-kernel >= 3.4.4`
- 启用 TLS 时额外需要 `galay-ssl` 与 OpenSSL

## 基础构建

下面命令对应根 `CMakeLists.txt` 的真实选项；它会生成库、示例、测试与 benchmark：

```bash
cmake -S . -B build \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_PREFIX_PATH=/Users/gongzhijie/Desktop/projects/git/.galay-prefix/latest \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_TESTING=ON \
  -DBUILD_BENCHMARKS=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build --parallel
```

## TLS 构建

如果你要使用 `HttpsServer`、`HttpsClient`、`WssClient`、`H2Server` 或 `H2Client`，必须在配置阶段启用 TLS：

```bash
cmake -S . -B build-ssl \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_PREFIX_PATH=/Users/gongzhijie/Desktop/projects/git/.galay-prefix/latest \
  -DGALAY_HTTP_ENABLE_SSL=ON \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_TESTING=ON \
  -DBUILD_BENCHMARKS=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build-ssl --parallel
```

示例证书可直接使用仓库自带的 `test/test.crt` 与 `test/test.key`。

## 第一个 HTTP 示例

源码与 target：

- 服务端：`examples/include/E1-echo_server.cpp` → `E1-EchoServer`
- 客户端：`examples/include/E2-echo_client.cpp` → `E2-EchoClient`

运行：

```bash
./build/examples/E1-EchoServer 8080
./build/examples/E2-EchoClient http://127.0.0.1:8080/echo "hello"
```

## 第一个 h2c 示例

源码与 target：

- 服务端：`examples/include/E9-h2c_echo_server.cpp` → `E9-H2cEchoServer`
- 客户端：`examples/include/E10-h2c_echo_client.cpp` → `E10-H2cEchoClient`

运行：

```bash
./build/examples/E9-H2cEchoServer 9080
./build/examples/E10-H2cEchoClient 127.0.0.1 9080
```

## 第一个 TLS 示例

源码与 target：

- HTTPS 服务端：`examples/include/E5-https_server.cpp` → `E5-HttpsServer`
- HTTPS 客户端：`examples/include/E6-https_client.cpp` → `E6-HttpsClient`

运行：

```bash
./build-ssl/examples/E5-HttpsServer 8443 test/test.crt test/test.key
./build-ssl/examples/E6-HttpsClient https://127.0.0.1:8443/
```

## 常用校验 target

测试与 benchmark target 名都直接继承真实文件名：

```bash
cmake --build build --target T26-h2c_server T25-h2c_client B1-HttpServer --parallel 4
cmake --build build-ssl --target T21-https_server T22-https_client T28-h2_server T29-h2_client --parallel 4
```
