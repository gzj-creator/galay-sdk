# 06-高级主题

## 协程并发约束

- 在 `Task` 协程函数中禁止使用阻塞同步原语（线程锁、条件变量等）。
- 协程内共享状态请使用 `AsyncMutex` / `UnsafeChannel` / `AsyncWaiter` 等协程友好原语。
- 临界区不要跨 `co_await`，先复制快照再执行异步发送/读写。

## 性能优化

### 静态文件传输优化

根据文件大小选择合适的传输模式：

```cpp
HttpRouter router;

// 小文件（< 1MB）：内存传输
StaticFileConfig memory_config;
memory_config.setTransferMode(FileTransferMode::MEMORY);
router.mount("/images", "./images", memory_config);

// 大文件：零拷贝传输
StaticFileConfig sendfile_config;
sendfile_config.setTransferMode(FileTransferMode::SENDFILE);
router.mount("/videos", "./videos", sendfile_config);

// 自动选择（推荐）
StaticFileConfig auto_config;
auto_config.setTransferMode(FileTransferMode::AUTO);
router.mount("/files", "./files", auto_config);
```

### Keep-Alive 连接复用

HTTP/1.1 默认启用 Keep-Alive，客户端可复用连接：

```cpp
Task<void> multipleRequests() {
    HttpClient client;
    auto connect_result = co_await client.connect("http://api.example.com:80");
    if (!connect_result) {
        co_return;
    }

    auto session = client.getSession();
    auto writer = session.getWriter();
    auto reader = session.getReader();

    // 复用同一连接发送多个请求
    auto req1 = Http1_1RequestBuilder::get("/api/v1/users").host("api.example.com").build();
    auto send1 = co_await writer.sendRequest(req1);
    if (!send1 || !send1.value()) {
        co_return;
    }
    HttpResponse resp1;
    auto recv1 = co_await reader.getResponse(resp1);
    if (!recv1 || !recv1.value()) {
        co_return;
    }

    auto req2 = Http1_1RequestBuilder::get("/api/v1/posts").host("api.example.com").build();
    auto send2 = co_await writer.sendRequest(req2);
    if (!send2 || !send2.value()) {
        co_return;
    }
    HttpResponse resp2;
    auto recv2 = co_await reader.getResponse(resp2);
    if (!recv2 || !recv2.value()) {
        co_return;
    }

    auto close_result = co_await client.close();
    if (!close_result) {
        co_return;
    }
}
```

## 中间件模式

> 本节讨论的是业务层组织方式；下面的例子只使用当前仓库真实存在的 `HttpConn` / `HttpRequest` / `Http1_1ResponseBuilder` 等公开类型，不引入仓库外的“中间件框架”假想 API。

### 日志中间件

```cpp
Task<void> loggingHandler(HttpConn& conn, HttpRequest req) {
    auto start = std::chrono::steady_clock::now();
    HTTP_LOG_INFO("[middleware] [{}] [{}]",
                  httpMethodToString(req.header().method()),
                  req.header().uri());

    auto response = Http1_1ResponseBuilder::ok()
        .text("logged")
        .build();
    auto writer = conn.getWriter();
    while (true) {
        auto result = co_await writer.sendResponse(response);
        if (!result || result.value()) {
            break;
        }
    }

    auto end = std::chrono::steady_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    HTTP_LOG_INFO("[middleware] [elapsed={}ms]", duration.count());
    co_return;
}
```

### 认证中间件

```cpp
Task<void> authGate(HttpConn& conn, HttpRequest req) {
    const std::string auth_header = req.header().headerPairs().getValue("Authorization");
    if (auth_header.empty()) {
        auto response = Http1_1ResponseBuilder::unauthorized()
            .json("{\"error\":\"Unauthorized\"}")
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(response);
            if (!result || result.value()) break;
        }
        co_return;
    }

    auto response = Http1_1ResponseBuilder::ok()
        .json("{\"authorized\":true}")
        .build();
    auto writer = conn.getWriter();
    while (true) {
        auto result = co_await writer.sendResponse(response);
        if (!result || result.value()) {
            break;
        }
    }
    co_return;
}
```

### 广播消息

```cpp
struct BroadcastManager {
    std::vector<WsWriter*> clients;
    AsyncMutex mutex;

    Task<void> addClient(WsWriter* writer) {
        auto lock_result = co_await mutex.lock();
        if (!lock_result) {
            co_return;
        }
        clients.push_back(writer);
        mutex.unlock();
        co_return;
    }

    Task<void> broadcast(const std::string& message) {
        auto lock_result = co_await mutex.lock();
        if (!lock_result) {
            co_return;
        }
        auto snapshot = clients;
        mutex.unlock();

        for (auto* client : snapshot) {
            auto send_result = co_await client->sendText(message);
            if (!send_result) {
                continue;
            }
        }
    }
};
```

### 发送路径降拷贝（建议）

`WsWriter/WsSession` 支持右值发送接口，业务侧可在不再复用 payload 时显式 `std::move`，
减少发送热路径中的字符串复制。

```cpp
std::string text = buildMessage();
co_await writer.sendText(std::move(text));

std::string bin = buildBinary();
co_await writer.sendBinary(std::move(bin));

WsFrame frame(WsOpcode::Text, std::string("hello"), true);
co_await writer.sendFrame(std::move(frame));
```

适用场景：
- Echo/转发后不再使用原消息内容
- 高 QPS 压测发送循环

## 错误处理策略

### 重试机制

```cpp
template<typename Func>
Task<std::optional<int>> retryWithBackoff(Func func, int max_retries = 3) {
    for (int i = 0; i < max_retries; ++i) {
        auto result = co_await func();
        if (result) co_return result;

        if (i < max_retries - 1) {
            auto delay = std::chrono::seconds(1 << i);  // 指数退避
            co_await galay::kernel::sleep(delay);
        }
    }
    co_return std::nullopt;
}
```

### 超时处理

```cpp
Task<void> requestWithTimeout() {
    HttpClient client;
    auto connect_result = co_await client.connect("http://api.example.com:80");
    if (!connect_result) {
        co_return;
    }

    auto session = client.getSession();
    auto writer = session.getWriter();
    auto reader = session.getReader();

    auto request = Http1_1RequestBuilder::get("/api/slow")
        .host("api.example.com")
        .build();

    // 发送请求
    auto send_result = co_await writer.sendRequest(request);
    if (!send_result || !send_result.value()) {
        co_return;
    }

    // 接收响应（带超时）
    HttpResponse response;
    auto read_awaitable = reader.getResponse(response);
    auto result = co_await read_awaitable.timeout(std::chrono::seconds(5));

    if (!result.has_value()) {
        std::cerr << "请求超时\n";
    }

    auto close_result = co_await client.close();
    if (!close_result) {
        co_return;
    }
}
```

### 速率限制

```cpp
struct RateLimiter {
    std::unordered_map<std::string, std::pair<int, std::chrono::steady_clock::time_point>> requests;
    AsyncMutex mutex;
    int max_requests = 100;
    std::chrono::seconds window = std::chrono::seconds(60);

    Task<void> allowRequest(const std::string& client_ip, bool& allowed) {
        auto lock_result = co_await mutex.lock();
        if (!lock_result) {
            allowed = false;
            co_return;
        }

        auto now = std::chrono::steady_clock::now();
        auto& [count, last_reset] = requests[client_ip];

        if (now - last_reset > window) {
            count = 0;
            last_reset = now;
        }

        if (count >= max_requests) {
            allowed = false;
            mutex.unlock();
            co_return;
        }

        ++count;
        allowed = true;
        mutex.unlock();
        co_return;
    }
};
```

## 与其他库集成

### 与 JSON 库集成

```cpp
#include <nlohmann/json.hpp>

using json = nlohmann::json;

Task<void> jsonApiHandler(HttpConn& conn, HttpRequest req) {
    try {
        json request_json = json::parse(req.getBodyStr());
        std::string name = request_json["name"];

        json response_json = {
            {"status", "success"},
            {"data", {{"name", name}}}
        };

        auto response = Http1_1ResponseBuilder::ok()
            .json(response_json.dump())
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(response);
            if (!result || result.value()) break;
        }

    } catch (const json::exception& e) {
        auto response = Http1_1ResponseBuilder::badRequest()
            .json("{\"error\":\"Invalid JSON\"}")
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(response);
            if (!result || result.value()) break;
        }
    }

    co_return;
}
```
